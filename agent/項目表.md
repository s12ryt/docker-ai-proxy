# 項目表 (項目結構與依賴)

## 項目簡介
AI Hub 是多模型聚合網關，支援多協定與多供應商，使用 Go 語言編寫，支持 SQLite/MySQL/PostgreSQL。

## 檔案結構
- `cmd/ai-hub/` : 主程式入口
- `internal/config/` : 配置載入 (env + json)
- `internal/providers/` : 密鑰輪轉
- `internal/proxy/` : 上游請求路由與轉發
- `internal/store/` : 持久化 (SQLite / MySQL / PostgreSQL)
- `internal/server/` : HTTP 路由、中間件、Web 端點
    - `internal/server/web/` : 玻璃擬態前端 (landing + dashboard)
- `agent/` : 管理檔案 (本專案)

## 核心依賴
- `go` (1.22+)
- `golang.org/x/crypto/bcrypt` (已用於密碼雜湊)
- Session token 目前使用 Go 標準庫 HMAC-SHA256 三段 JWT-like token；未加入 `github.com/golang-jwt/jwt/v5`，因本機環境找不到 `go` 無法 `go get`

## 資料庫驅動 (現有)
- `modernc.org/sqlite` (純 Go SQLite)
- `github.com/go-sql-driver/mysql`
- `github.com/jackc/pgx/v5/stdlib`
