# 項目表 · docker-ai-proxy

> 紀錄這個專案各個目錄/檔案幹嘛、第三方依賴、CI workflow 用途。
> 接手新會話前先讀這份。

## 一、技術棧

| 層 | 選型 | 備註 |
|---|---|---|
| 語言 | Go 1.22 | 純靜態，CGO_ENABLED=0 |
| HTTP | 標準庫 `net/http` + `http.ServeMux` | 沒用 chi/gin |
| 資料庫 | SQLite (`modernc.org/sqlite`) / MySQL (`go-sql-driver/mysql`) / PostgreSQL (`jackc/pgx/v5/stdlib`) 三選一 | 全純 Go 驅動，無 CGO；由 `DB_DRIVER` 切換 |
| 前端 | 原生 HTML/JS/CSS，embed.FS | 暗黑玻璃擬態 (landing + dashboard) |
| 容器 | 多階段 distroless | 鏡像 ~20–25 MB |
| CI/CD | GitHub Actions | `ci.yml` / `docker-publish.yml` / `release.yml` |

## 三、依賴

```
github.com/s12ryt/docker-ai-proxy
├── modernc.org/sqlite v1.34.1            # 純 Go SQLite，預設 driver
├── github.com/go-sql-driver/mysql v1.8.1 # 純 Go MySQL driver，DB_DRIVER=mysql 時啟用
└── github.com/jackc/pgx/v5 v5.7.1        # 純 Go PostgreSQL driver (stdlib mode)，DB_DRIVER=postgres 時啟用
```

三個 driver 透過 `database/sql` 與 `internal/store` 的 dialect 抽象一起運作；不會編到 CGO。`go.sum` 由 CI（與 Dockerfile builder）的 `go mod tidy` 自動補齊。

## 四、環境變數

| 變數 | 預設 | 說明 |
|---|---|---|
| `LISTEN` | `:8080` | 監聽 |
| `ADMIN_TOKEN` | `change-me-admin` | 控制台 + `/api/*` |
| `ACCESS_TOKENS` | (空) | 逗號分隔；空 = 開放 `/v1/*` |
| `TELEGRAM_USER_ID` | (空) | Telegram 使用者 ID（預留給通知/審計整合） |
| `TELEGRAM_BOT_ID` | (空) | Telegram Bot ID（預留給通知/審計整合） |
| `DB_DRIVER` | `sqlite` | `sqlite` / `mysql` / `postgres` 三選一 |
| `DB_PATH` | `data/ai-hub.db` | 僅 `DB_DRIVER=sqlite` 生效 |
| `DB_DSN` | (空) | `mysql` 與 `postgres` 必填的連線字串 |
| `DB_MAX_OPEN_CONNS` | sqlite=1, 雲端=10 | sql.DB.SetMaxOpenConns |
| `DB_MAX_IDLE_CONNS` | sqlite=1, 雲端=5 | sql.DB.SetMaxIdleConns |
| `DB_CONN_MAX_LIFETIME` | 雲端=`30m`,sqlite 留空 | Go duration 字串 (透過 `time.ParseDuration`) |
| `CONFIG_PATH` | `config.json` | 檔案配置路徑 |
| `ENABLE_METRICS` | `1` | 是否寫入呼叫日誌 |

優先級：env > config.json > defaults。

## 五、CI 行為摘要

- `ci.yml`：gofmt-check → go vet → `go test -race -coverprofile=coverage.out -covermode=atomic` (註：先前因 race+atomic 衝突已調整為非 atomic) → docker buildx 驗證 amd64+arm64。
- `docker-publish.yml`：push main / tag → buildx push 到 `ghcr.io/s12ryt/docker-ai-proxy:{latest,sha-...,v...}` + SBOM + provenance。
- `release.yml`：tag v*.*.* → cross-compile binaries → GitHub Release attach。
